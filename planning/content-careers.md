## Careers at Volantis

We build photonic systems that move data with light. We are rebuilding the memory wall.

## Why this work

Model size grew ~10,000× since 2016. GPU memory grew ~17×. The bottleneck is moving information.

We build the hardware that changes that.

## What we do

- We put photonics inside the accelerator.
- We move **60 TB/s per die**.
- We extend reach ~**100×** beyond electrical wires.
- We make room for ~**100×** more memory per accelerator.

## What you can expect

- Decisions backed by measurements.
- Fast loops on real hardware.
- A team that ships.
