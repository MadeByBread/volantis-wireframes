# Homepage

**Purpose:** Communicate the value prop in seconds, establish credibility, route visitors to Product / Tech / About.
**Primary audience:** NeoCloud buyers + investors (mixed top-of-funnel).
**Primary CTA:** Talk to sales / Reserve capacity. **Secondary:** Explore the tech.

**SEO**

- **Title:** Volantis — Photonic Interconnect for Large-Model AI Inference
- **Meta description:** Volantis builds photonic wires that pack up to 100× more memory into an accelerator — delivering >30× lower latency, **15× Tok/$**, and **6× Tok/W** for 1T+ models.
- **H1:** Big models at low latency — finally affordable.

**Section outline**

1. **Hero**
    - H1 with the concrete promise (memory + latency + cost), not the riddle.
    - One-line subhead: who it's for ("Built for NeoClouds running 1T+ MoE models").
    - Primary CTA button + secondary text link to Tech.
    - Keep the brand visual/gradient — strong and distinctive; just anchor it with a clear headline.
2. **Proof bar (social proof, immediately under hero)**
    - Backed by Sam Altman & Alex Wang · team from Micron, Intel, Broadcom, Ayar Labs, AMD, Qualcomm, Groq, MIT.
    - Logo strip.
3. **The problem (the wedge)**
    - "Model size has grown 10,000× while GPU memory has grown ~17×."
    - "The bottleneck is moving information, not processing it." (Keep this line — it's the best sentence on the current site.)
    - "Electrical wires only reach ~2mm, capping memory per chip."
4. **The solution**
    - Introduce the Volantis Photonic Motherboard / photonic wires in one tight paragraph.
    - 3 headline stats as scannable cards: **100× memory reach** · **>30× lower latency** · **15× Tok/$, 6× Tok/W**.
    - CTA to Tech ("How it works").
5. **Performance snapshot**
    - One benchmark chart: Volantis vs NVIDIA B200 on tok/s/user (1,590 vs ~50 on DeepSeek-V3.2; or 162 vs ~4–5 on GPT-1.8T).
    - Footnote with methodology (in/out=1024/1024, BS=32, fp8) so the claim reads as rigorous, not marketing.
    - CTA to Product.
6. **Who it's for**
    - Short statement targeting NeoCloud providers: capacity, economics, deployment.
7. **Closing CTA band**
    - Replace the repeated "Are you in?" loop with one decisive band: "Reserve capacity for [next available window]" + sales contact.
    - Secondary: Investor inquiries link.
8. **Footer** (site-wide) — working social links, real copyright year, nav, legal, careers, investor contact.
